# Challenge 2:  Continuar con mi primera automatización mobile

El propósito de este ejercicio era crear un proyecto Maven desde cero que pueda abrir una app tanto en un dispositivo real como en un emulador Android.

---

###  My Demo App (Android)

He utilizado para ello My Demo App, una aplicación Android diseñada para practicar automatización mobile con flujos reales.

---

### Instrucciones para el ejercicio

1. Crear un proyecto Maven desde cero en IntelliJ IDEA.
2. Agregar las dependencias mínimas necesarias para Appium en el archivo `pom.xml
3. Crear una clase de prueba básica que pueda abrir la app `mda-2.2.0-25.apk` en ambos dispositivos (emulador y real).

### ⚠️ Nota QA Automation:
No se pueden ejecutar pruebas en un dispositivo real debido a la incompatibilidad de Vysor con la versión de Android de mi dispositivo, por lo que las pruebas solo se harán en el Emulador.
